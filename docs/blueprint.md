# **App Name**: LearnStream

## Core Features:

- Course Creation & Management: Admins can create, edit, organize, and manage course content including lessons, modules, and resources (text, video links).
- User Authentication & Profiles: Users can register, log in, and manage their personal profiles. This includes tracking course progress and subscription status.
- Free Course Access: Authenticated users can browse and access designated free courses without requiring an active subscription.
- Subscription Gateway Integration: Integrate a payment gateway (e.g., Stripe) to handle subscription payments and manage user access to premium content.
- Interactive Course Player: A responsive interface for users to consume course content, including video playback, text display, and progress tracking.
- AI Lesson Assistant: An AI tool that can generate concise summaries of lesson content or answer specific questions posed by students, drawing insights directly from the course material.

## Style Guidelines:

- A sophisticated deep blue-violet as the primary color: #4426D9. This hue evokes trust, depth of knowledge, and modern digital professionalism.
- A serene and lightly desaturated background color, derived from the primary hue, to ensure content legibility: #F1F0F4.
- A brighter, softer blue for accent elements and interactive components: #85A4E0. It provides contrast and a sense of calm engagement.
- Headline font: 'Space Grotesk' (sans-serif) for a contemporary and structured feel. Body font: 'Inter' (sans-serif) for excellent readability in longer educational texts.
- Utilize a consistent set of minimal, outline-style icons. These will convey functionality clearly without distracting from educational content.
- Implement a clean, spacious, and grid-based layout for intuitive navigation and presentation of course materials. Ensure responsiveness for seamless viewing across all device sizes.
- Incorporate subtle and smooth transitions for page loading, module navigation, and interactive elements, enhancing user flow without being intrusive.
